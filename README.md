# TRIO SIGNO BACKEND

## Description

Back-end of Triosigno app using [Nest](https://github.com/nestjs/nest) with [Prisma](https://www.prisma.io/) and [PostgreSQL](https://www.postgresql.org/). All with [TypeScript](https://www.typescriptlang.org/).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
